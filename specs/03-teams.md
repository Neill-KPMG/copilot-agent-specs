# Teams

## Purpose
Send and read chat or channel messages, manage chats, and discover teams/channels.

## Core Activities
- Post new messages to a person, group chat, or channel
- Read and summarize recent chat or channel activity
- Reply in threads and mention specific people
- Create 1:1 or group chats and add members
- Create new channels (standard or private)
- Attach files to messages
- Update or delete your own messages

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Action** | _(fill in)_ | Post / reply / read / summarize / create chat / create channel |
| **Recipient** | _(fill in)_ | Person name(s), group chat topic, or Team + channel name |
| **Topic / message content** | _(fill in)_ | What the message is about or what to look for |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Tone** | _(fill in)_ | Casual, professional, urgent, celebratory, etc. |
| **Subject line** | _(fill in)_ | Headline shown above the message body (channel posts) |
| **Mentions** | _(fill in)_ | @-mentions of specific people |
| **Attachments** | _(fill in)_ | Files to include with the message |
| **Reply target** | _(fill in)_ | Specific message to reply to, if threaded |

## Example

> Post to the Engineering channel: we're shipping the API fix today at 4pm PT — casual tone, include the runbook link.

## Expected Output
Message sent, or summary of channel/chat activity returned in chat.
