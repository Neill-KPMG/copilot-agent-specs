# Document Creation — PDF

## Purpose
Generate, transform, and manipulate PDFs: merge, split, watermark, OCR, fill forms, extract content.

## Core Activities
- Create PDFs from scratch or from other documents
- Merge multiple PDFs into one
- Split or extract specific page ranges
- Add watermarks, rotate pages, encrypt with password
- Extract text and tables, including OCR for scanned documents
- Fill out form fields programmatically
