# Files (SharePoint / OneDrive)

## Purpose
Find, open, read, and summarize content across your OneDrive and SharePoint sites.

## Core Activities
- Search OneDrive and SharePoint by keyword, file name, or topic
- Browse drives, folders, and document libraries
- Read file contents (Word, Excel, PDF, text, CSV)
- Get file metadata (owner, modified date, size)
- Create folders and navigate subsites
- Read SharePoint list items and columns

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Action** | _(fill in)_ | Find / read / summarize / extract / list |
| **File identifier** | _(fill in)_ | File name, type, or topic keyword |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Location hint** | _(fill in)_ | My OneDrive, specific SharePoint site, shared with me |
| **Specific section** | _(fill in)_ | Sheet/tab name, page range, heading, named range |
| **Purpose** | _(fill in)_ | Why you need it — sharpens summary and extraction |
| **Output format** | _(fill in)_ | Bullet summary, table extraction, key numbers only, raw text |

## Example

> Find the Q2 forecast spreadsheet in my OneDrive, read the Summary tab, and tell me the variance vs plan.

## Expected Output
File contents, summary, or extracted data returned inline.
