# Visual Rendering (Adaptive Cards)

## Purpose
Turn structured data into rich visual cards: dashboards, charts, tables, KPIs.

## Core Activities
- Render dashboards with KPI tiles, charts, and tables
- Visualize trends, comparisons, and part-of-whole data
- Build status reports with color coding and progress indicators
- Display tabular data with sorting and grouping
- Combine multiple views in a single card

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Data or topic** | _(fill in)_ | What to visualize — the underlying dataset or summary |
| **View type** | _(fill in)_ | Dashboard, chart (bar/line/pie/gauge), table, KPI, status |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Layout preference** | _(fill in)_ | Single chart, KPI tiles + chart, grid, stacked |
| **Color coding** | _(fill in)_ | Brand colors, red/yellow/green status, monochrome |
| **Sort order** | _(fill in)_ | Highest first, alphabetical, chronological |
| **Metric labels** | _(fill in)_ | How metrics should be named and formatted |
| **Click-through actions** | _(fill in)_ | Buttons or links to add |
