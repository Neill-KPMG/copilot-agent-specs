# Skills & Customization

## Purpose
Personalize how Copilot works for you — create, edit, view, or delete personal skills.

## Core Activities
- Create new personal skills with triggers and behaviors
- View and list all your existing skills
- Edit or update skill definitions
- Delete skills you no longer want
- Tune personal instructions that apply across all sessions
- Validate and audit skills before use

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Action** | _(fill in)_ | Create / edit / view / delete / validate / audit |
| **Skill name or purpose** | _(fill in)_ | What the skill does, in one short line |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Triggers / keywords** | _(fill in)_ | Phrases that should activate the skill |
| **Tool dependencies** | _(fill in)_ | Which tools the skill needs to call |
| **Output behavior** | _(fill in)_ | How the result should be formatted or delivered |
| **Examples** | _(fill in)_ | Sample prompts that should match this skill |
| **Scope** | _(fill in)_ | Personal (only you) or shared (your team) |

## Example

> Create a skill called 'standup-notes' that formats my daily standup as bullets: yesterday, today, blockers.

## Expected Output
Skill created, updated, or removed; confirmation message returned.
