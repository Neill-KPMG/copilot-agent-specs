# Document Creation — PowerPoint

## Purpose
Design presentations (.pptx): pitch decks, status reviews, board updates, and briefings.

## Core Activities
- Create new decks from scratch or from a template
- Apply design guidance: color palette, typography, layout
- Insert charts, diagrams, images, and speaker notes
- Read and edit existing decks
- Run visual QA via slide-to-image conversion

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Deck purpose** | _(fill in)_ | What the presentation is for |
| **Audience** | _(fill in)_ | Who will see it — board, customers, team, investors |
| **Slide count or outline** | _(fill in)_ | Number of slides or slide-by-slide outline |
| **Style / template** | _(fill in)_ | Existing template, or style preferences |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Data sources** | _(fill in)_ | Numbers, charts, or source documents to draw from |
| **Branding** | _(fill in)_ | Logo, colors, company template |
| **Speaker notes** | _(fill in)_ | Required yes/no, level of detail |
| **Visual style** | _(fill in)_ | Photo-heavy, text-heavy, diagram-driven, etc. |
| **Charts / diagrams** | _(fill in)_ | Specific visuals to include |
| **Images** | _(fill in)_ | Stock images yes/no, subjects, mood |

## Example

> Make a 10-slide investor update for our Series A board covering traction, revenue, hiring, and ask; clean modern style; numbers from the Q1 metrics doc.

## Expected Output
A .pptx file saved to your output folder, ready to download.
