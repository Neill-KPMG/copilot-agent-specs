# Document Creation — Excel

## Purpose
Build spreadsheets (.xlsx): trackers, financial models, data analyses, and multi-sheet workbooks.

## Core Activities
- Create new workbooks with multiple sheets, formulas, and formatting
- Build financial models with industry-standard color coding
- Add charts, pivot tables, and conditional formatting
- Read, edit, and analyze existing spreadsheets
- Clean data and recalculate formulas via LibreOffice

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Workbook purpose** | _(fill in)_ | What the spreadsheet is for |
| **Sheet / tab list** | _(fill in)_ | Names of each tab to include |
| **Columns per sheet** | _(fill in)_ | Column headers and data types |
| **Data source** | _(fill in)_ | Where data comes from — file, query, manual, etc. |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Formulas / calculations** | _(fill in)_ | Specific formulas, KPIs, or derived columns |
| **Charts** | _(fill in)_ | Chart types per sheet (bar, line, pie, etc.) |
| **Formatting** | _(fill in)_ | Color coding, number formats, conditional rules |
| **Pivot tables** | _(fill in)_ | Pivot definitions (rows, columns, values) |
| **Validation rules** | _(fill in)_ | Dropdowns, ranges, required fields |
