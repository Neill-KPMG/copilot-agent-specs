# Email (Outlook)

## Purpose
Search, summarize, draft, send, reply, forward, and triage messages across your inbox and folders.

## Core Activities
- Search across mail by sender, topic, date, attachments, or flag status
- Summarize threads and long messages into key points and action items
- Draft and send new emails, replies, reply-alls, and forwards
- Save drafts, update them, and add attachments before sending
- Move, flag, mark read/unread, categorize, and delete messages
- Create folders and set up inbox rules (auto-move, forward, mark)
- Manage out-of-office and automatic replies

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Action** | _(fill in)_ | search / summarize / draft / send / reply / forward / triage |
| **Target (sender or topic)** | _(fill in)_ | Person name, email address, keyword, or thread reference |
| **Time window** | _(fill in)_ | e.g. last 7 days, this month, since Monday, specific date range |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Recipient(s)** | _(fill in)_ | Names or email addresses for outgoing messages |
| **Tone** | _(fill in)_ | Formal, casual, friendly, direct, apologetic, etc. |
| **Format** | _(fill in)_ | Bullet points, narrative paragraph, table, etc. |
| **Attachments** | _(fill in)_ | Files to include |
| **Priority / flag** | _(fill in)_ | High importance, follow-up flag, etc. |
| **Folder scope** | _(fill in)_ | Inbox, Sent, Archive, custom folder name |

## Example

> Summarize emails from Sarah in the last 7 days about the Q3 launch — bullet points, flag anything needing a reply.

## Expected Output
Summary in chat, or a sent/drafted email saved to Sent Items or Drafts.
