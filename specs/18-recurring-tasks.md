# Recurring / Scheduled Tasks

## Purpose
Automate work to run on a schedule — daily digests, weekly reports, end-of-day recaps.

## Core Activities
- Set up daily, weekly, monthly, or custom-interval tasks
- Schedule one-time tasks for a future date/time
- Pause or edit existing scheduled tasks
- Run inline (preserves context across runs) or separate (fresh each time)

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Recurrence** | _(fill in)_ | Daily, weekly, monthly, every weekday, every N hours, one-time |
| **Time of day** | _(fill in)_ | Specific hour and minute (in your local time zone) |
| **Task description** | _(fill in)_ | What to do on each run — written as a standalone instruction |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Delivery channel** | _(fill in)_ | Email, Teams DM, inline |
| **Start date** | _(fill in)_ | When to begin running |
| **End date** | _(fill in)_ | When to stop, if applicable |
| **Pause conditions** | _(fill in)_ | Holidays, weekends, custom |
| **Output format** | _(fill in)_ | Brief summary, full report, table, bullets |

## Example

> Every weekday at 8am, send me a Teams DM with my top 3 priority emails and today's first meeting.

## Expected Output
Scheduled task created and confirmed; runs automatically per schedule.
