# Image Search

## Purpose
Find public-license images for use in slides, documents, and emails.

## Core Activities
- Search royalty-free images by keyword
- Return multiple options with attribution
- Drop directly into slides, documents, or emails as visuals
- Filter by style (photo, illustration, diagram)

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Subject** | _(fill in)_ | What the image should depict |
| **Usage** | _(fill in)_ | Slide, document, email, web |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
