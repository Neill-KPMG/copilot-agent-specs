# Document Creation — Word

## Purpose
Produce polished Word documents (.docx): memos, reports, letters, briefs, and more.

## Core Activities
- Create new Word documents from scratch or from your specs
- Apply professional formatting: TOC, headings, page numbers, headers/footers
- Insert tables, lists, images, and tracked changes
- Edit existing Word documents and accept/reject changes
- Export to PDF when needed

## Required Fields

| Field | Your Input | Notes |
|---|---|---|
| **Document type** | _(fill in)_ | Memo, report, letter, brief, proposal, policy, etc. |
| **Audience** | _(fill in)_ | Who will read it — executives, customers, internal team, regulators |
| **Key sections** | _(fill in)_ | Outline of headings / topics to cover |
| **Tone** | _(fill in)_ | Formal, conversational, persuasive, technical, etc. |

## Optional Fields

| Field | Your Input | Notes |
|---|---|---|
| **Source material** | _(fill in)_ | Emails, files, transcripts, data points to draw from |
| **Length** | _(fill in)_ | Page count, word count, or 'concise' / 'comprehensive' |
| **Branding / header** | _(fill in)_ | Company name, logo, letterhead requirements |
| **Tracked changes** | _(fill in)_ | Required yes/no, and author name to use |
| **Tables / figures** | _(fill in)_ | Specific tables, charts, or images to include |
| **Style template** | _(fill in)_ | Existing template to match, or style preferences |
