# Contributing

## Propose a new capability area

1. Open an issue using the **New capability area** template.
2. Once approved, open a PR that adds:
   - `specs/NN-<slug>.md` — full specification
   - `templates/NN-<slug>.md` — prompt template
   - A new row in the README capability index table

## Refine an existing spec

1. Open an issue using the **Refine existing spec** template.
2. Open a PR with the proposed change — keep changes focused on one area per PR.

## Style guide

- Required fields = must be present for the agent to act
- Optional fields = improve quality but not strictly necessary
- Examples should be fully completed prompts, not partial fragments
- Keep tone neutral and instructive — these are reference materials
