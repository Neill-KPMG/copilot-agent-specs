# Prompt Template — Teams

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Post / reply / read / summarize / create chat / create channel>
[RECIPIENT]: <Person name(s), group chat topic, or Team + channel name>
[TOPIC / MESSAGE CONTENT]: <What the message is about or what to look for>

OPTIONAL:
- Tone: <Casual, professional, urgent, celebratory, etc.>
- Subject line: <Headline shown above the message body (channel posts)>
- Mentions: <@-mentions of specific people>
- Attachments: <Files to include with the message>
- Reply target: <Specific message to reply to, if threaded>
```

## Worked example
Post to the Engineering channel: we're shipping the API fix today at 4pm PT — casual tone, include the runbook link.
