# Prompt Template — Document Creation — PDF

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Create / merge / split / extract / watermark / OCR / fill form>
[SOURCE FILE(S)]: <PDF(s) or other documents to act on>

OPTIONAL:
- Page range: <e.g. pages 1–5, last 10, odd/even only>
- Watermark text: <Text and position (e.g. CONFIDENTIAL, diagonal)>
- Output filename: <Preferred file name>
- Form fields: <Specific fields to fill and their values>
- Password / encryption: <If output should be password-protected>
```
