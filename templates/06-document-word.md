# Prompt Template — Document Creation — Word

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Memo, report, letter, brief, proposal, policy, etc.>
[AUDIENCE]: <Who will read it — executives, customers, internal team, regulators>
[KEY SECTIONS]: <Outline of headings / topics to cover>
[TONE]: <Formal, conversational, persuasive, technical, etc.>

OPTIONAL:
- Source material: <Emails, files, transcripts, data points to draw from>
- Length: <Page count, word count, or 'concise' / 'comprehensive'>
- Branding / header: <Company name, logo, letterhead requirements>
- Tracked changes: <Required yes/no, and author name to use>
- Tables / figures: <Specific tables, charts, or images to include>
- Style template: <Existing template to match, or style preferences>
```

## Worked example
Create a project status memo for the steering committee, covering progress, risks, and asks, formal tone, using the data from last week's Teams thread.
