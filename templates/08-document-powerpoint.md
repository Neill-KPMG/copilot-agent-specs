# Prompt Template — Document Creation — PowerPoint

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <What the presentation is for>
[AUDIENCE]: <Who will see it — board, customers, team, investors>
[SLIDE COUNT OR OUTLINE]: <Number of slides or slide-by-slide outline>
[STYLE / TEMPLATE]: <Existing template, or style preferences>

OPTIONAL:
- Data sources: <Numbers, charts, or source documents to draw from>
- Branding: <Logo, colors, company template>
- Speaker notes: <Required yes/no, level of detail>
- Visual style: <Photo-heavy, text-heavy, diagram-driven, etc.>
- Charts / diagrams: <Specific visuals to include>
- Images: <Stock images yes/no, subjects, mood>
```

## Worked example
Make a 10-slide investor update for our Series A board covering traction, revenue, hiring, and ask; clean modern style; numbers from the Q1 metrics doc.
