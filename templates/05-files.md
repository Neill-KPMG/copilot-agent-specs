# Prompt Template — Files (SharePoint / OneDrive)

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Find / read / summarize / extract / list>
[FILE IDENTIFIER]: <File name, type, or topic keyword>

OPTIONAL:
- Location hint: <My OneDrive, specific SharePoint site, shared with me>
- Specific section: <Sheet/tab name, page range, heading, named range>
- Purpose: <Why you need it — sharpens summary and extraction>
- Output format: <Bullet summary, table extraction, key numbers only, raw text>
```

## Worked example
Find the Q2 forecast spreadsheet in my OneDrive, read the Summary tab, and tell me the variance vs plan.
