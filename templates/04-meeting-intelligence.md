# Prompt Template — Meeting Intelligence

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Prep / summarize / extract / recap>
[MEETING REFERENCE]: <Meeting title, date/time, or 'most recent'>

OPTIONAL:
- Output focus: <Decisions only, action items only, attendee insights, key quotes>
- Time range: <If scanning multiple meetings>
- Related context: <Pull in related emails, files, or prior meeting notes>
- Audience for brief: <Who will read the brief — adjusts level of detail>
```

## Worked example
Prep me for tomorrow's 9am budget review — pull the last meeting's action items, attendee roles, and related emails from finance.
