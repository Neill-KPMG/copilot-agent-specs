# Prompt Template — Image Search

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <What the image should depict>
[USAGE]: <Slide, document, email, web>

OPTIONAL:
- Style: <Photo, illustration, diagram, vector>
- Quantity: <How many options to return>
- Orientation: <Landscape, portrait, square>
- Color palette: <Specific colors or mood (warm, cool, monochrome)>
- Mood / tone: <Professional, energetic, calm, inspiring>
```

## Worked example
Find 3 royalty-free photos of modern open-plan offices for a slide background.
