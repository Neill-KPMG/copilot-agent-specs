# Prompt Template — Document Creation — Excel

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <What the spreadsheet is for>
[SHEET / TAB LIST]: <Names of each tab to include>
[COLUMNS PER SHEET]: <Column headers and data types>
[DATA SOURCE]: <Where data comes from — file, query, manual, etc.>

OPTIONAL:
- Formulas / calculations: <Specific formulas, KPIs, or derived columns>
- Charts: <Chart types per sheet (bar, line, pie, etc.)>
- Formatting: <Color coding, number formats, conditional rules>
- Pivot tables: <Pivot definitions (rows, columns, values)>
- Validation rules: <Dropdowns, ranges, required fields>
```

## Worked example
Build a hiring tracker with tabs for Open Roles, Candidates, Offers; columns name, stage, recruiter, target start; pull data from the attached CSV.
