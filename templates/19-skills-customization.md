# Prompt Template — Skills & Customization

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Create / edit / view / delete / validate / audit>
[SKILL NAME OR PURPOSE]: <What the skill does, in one short line>

OPTIONAL:
- Triggers / keywords: <Phrases that should activate the skill>
- Tool dependencies: <Which tools the skill needs to call>
- Output behavior: <How the result should be formatted or delivered>
- Examples: <Sample prompts that should match this skill>
- Scope: <Personal (only you) or shared (your team)>
```

## Worked example
Create a skill called 'standup-notes' that formats my daily standup as bullets: yesterday, today, blockers.
