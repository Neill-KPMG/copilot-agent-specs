# Prompt Template — Email (Outlook)

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <search / summarize / draft / send / reply / forward / triage>
[TARGET (SENDER OR TOPIC)]: <Person name, email address, keyword, or thread reference>
[TIME WINDOW]: <e.g. last 7 days, this month, since Monday, specific date range>

OPTIONAL:
- Recipient(s): <Names or email addresses for outgoing messages>
- Tone: <Formal, casual, friendly, direct, apologetic, etc.>
- Format: <Bullet points, narrative paragraph, table, etc.>
- Attachments: <Files to include>
- Priority / flag: <High importance, follow-up flag, etc.>
- Folder scope: <Inbox, Sent, Archive, custom folder name>
```

## Worked example
Summarize emails from Sarah in the last 7 days about the Q3 launch — bullet points, flag anything needing a reply.
