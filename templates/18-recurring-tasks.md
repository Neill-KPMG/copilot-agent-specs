# Prompt Template — Recurring / Scheduled Tasks

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <Daily, weekly, monthly, every weekday, every N hours, one-time>
[TIME OF DAY]: <Specific hour and minute (in your local time zone)>
[TASK DESCRIPTION]: <What to do on each run — written as a standalone instruction>

OPTIONAL:
- Delivery channel: <Email, Teams DM, inline>
- Start date: <When to begin running>
- End date: <When to stop, if applicable>
- Pause conditions: <Holidays, weekends, custom>
- Output format: <Brief summary, full report, table, bullets>
```

## Worked example
Every weekday at 8am, send me a Teams DM with my top 3 priority emails and today's first meeting.
