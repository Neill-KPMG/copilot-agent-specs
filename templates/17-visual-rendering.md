# Prompt Template — Visual Rendering (Adaptive Cards)

Copy this template and fill in the brackets, then send it to Copilot.

```
[ACTION]: <What to visualize — the underlying dataset or summary>
[VIEW TYPE]: <Dashboard, chart (bar/line/pie/gauge), table, KPI, status>

OPTIONAL:
- Layout preference: <Single chart, KPI tiles + chart, grid, stacked>
- Color coding: <Brand colors, red/yellow/green status, monochrome>
- Sort order: <Highest first, alphabetical, chronological>
- Metric labels: <How metrics should be named and formatted>
- Click-through actions: <Buttons or links to add>
```

## Worked example
Render my Q1 team metrics as a dashboard with 3 KPI tiles on top and a trend chart below.
